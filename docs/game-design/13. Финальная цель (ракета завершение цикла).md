# Финальная цель (ракета завершение цикла)

Финальная цель (ракета / завершение цикла)

### 13.1. Определение

**Финальная цель — это конечный этап текущего игрового цикла, который требует:**
- максимальной оптимизации всех систем
- взаимодействия всех эпох
- полного использования производственных цепочек

### 13.2. Сущность цели

```text
FinalObjective = "Построить и запустить ракету"
```

### 13.3. Условия запуска ракеты

**Ракета требует:**

#### 13.3.1. Ресурсы всех эпох

resources from: stone age, industrial age, space age

#### 13.3.2. Продвинутые компоненты

сложные материалы

энергия

компоненты сборки

#### 13.3.3. Технологическое завершение

все критические технологии исследованы

#### 13.3.4. Производственная готовность

стабильный выпуск всех ресурсов

отсутствие bottlenecks (или их контролируемость)

#### 13.3.5. Достаточное количество рабочих циклов

```text
globalWorkCycles >= finalLaunchCycleThreshold
```

Ракета требует не только ресурсов, но и доказательства, что все эпохи работают стабильно и эффективно. `globalWorkCycles` — это и есть такое доказательство.

### 13.4. Этапы строительства ракеты

Этап 1: Каркас

базовые материалы

строительные ресурсы

Этап 2: Системы

электроника

механизмы

энергия

Этап 3: Топливо

высокоуровневые ресурсы

кросс-эпохные компоненты

Этап 4: Сборка

финальная интеграция всех систем

Этап 5: Запуск

trigger end-state sequence

### 13.5. Механика запуска

**После выполнения условий:**
- launch_button becomes active
- Игрок инициирует запуск вручную.

### 13.6. Награда за запуск

#### 13.6.1. Престиж

prestige_points += large_bonus (рассчитывается от globalWorkCycles)

#### 13.6.2. Метасистема

открытие новых глобальных улучшений

усиление кросс-эпохных связей

#### 13.6.3. Сброс цикла

reset world state (ресурсы, здания, технологии)

**Не сбрасывается:**
- globalWorkCycles
- накопленные очки престижа
- купленные постоянные бонусы

### 13.7. Престиж после ракеты

**После запуска:**
- игра возвращается в начало
- но с усилениями
- открываются новые уровни сложности

### 13.8. Мотивационный цикл

build -> optimize -> launch -> reset -> faster build

### 13.9. Баланс финальной цели

**Финальная цель должна:**
- быть долгой
- требовать всех систем
- не быть "одним кликом"
- но быть достижимой

### 13.10. Ошибки проектирования

Ракета слишком проста

Ракета не требует всех эпох

Отсутствие подготовки

Нет смысла в поздней игре

### 13.11. Минимальный набор (definition of done)

**Система реализована, если:**
- есть многоэтапная цель
- требуется кросс-эпохное производство
- есть финальный триггер запуска
- есть награда и престиж
- есть полный сброс прогресса
